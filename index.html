SOP & Governance

Jobdesk & Tata Kelola Data

1. Pembagian Jobdesk & Hak Akses

Pengelolaan data dibagi menjadi beberapa level akses demi keamanan:

▪ Azriel & Noura (Super Admin - Akses: Editor/Full Control)
  • Mengelola Master Database (File A) dan memastikan sistem Mirroring ke File B lancar.
  • Otomatisasi Real-time Alert (WA) dan Backup Plan harian (00.00 WIB).
  • Screening, validasi berkas, dan memegang kendali penuh status data.
▪ Ketua Pelaksana & Bendahara (Management - Akses: View Only Seluruh Data)

Memantau perkembangan jumlah pendaftar harian. Bendahara melakukan cross-check uang masuk berdasarkan data validasi tim.

▪ Koordinator Lomba (User - Akses: View Only Terbatas)

Hanya bisa melihat data khusus cabang lombanya sendiri melalui File B (Importrange) untuk teknis pertandingan.

2. Alur Kerja Harian (SOP Step-by-Step)
[GForm] ➔ [WA Alert] ➔ [Ronda Screening] ➔ [Assign Status] ➔ [File B]
  1. Entry & Alert: Peserta daftar via Google Form. Notifikasi otomatis langsung masuk ke grup WA Tim Pendataan.
  2. Standardisasi Format: Cek penulisan Nama (Awal Kata Kapital), No. Telp (Kode negara 62/20 tanpa "+"), dan Berkas (Format PDF).
  3. Ronda Screening (Ceklis Validasi): Cek keaslian Nama, cek nominal bukti bayar (apakah ada kode unik) & keaslian struk, serta pastikan link Drive berkas tidak dikunci.
  4. Sistem Flagging: Jika aman, ganti warna baris Spreadsheet jadi Hijau (Verified). Jika berkas bermasalah, tandai Merah dan admin segera hubungi peserta via WA.
3. Protokol Keamanan & Sistem Cadangan

▪ Prinsip Satu Pintu: Data masuk hanya via satu GForm resmi. Data keluar dikunci menggunakan rumus =IMPORTRANGE (File B - View Only).

▪ Komitmen NDA Internal: Azriel dan Noura berkomitmen menjaga kerahasiaan data. Dilarang keras mengunduh data ke perangkat pribadi tanpa izin.

▪ Mitigasi Risiko: Data diekspor berkala setiap pukul 00.00 WIB ke cloud cadangan demi menghindari kehilangan akibat masalah akun.

4. Timeline Kerja & Sinkronisasi

▪ Setiap Hari (23.00 WIB): Update total pendaftar harian ke grup inti panitia.

▪ 19 April: Sinkronisasi data awal dengan Divisi Bendahara untuk rekap uang masuk.

▪ 05 Mei: Serah terima data final 100% (Locked) ke Divisi Lomba untuk bagan pertandingan.

▪ 07 Mei: Penyerahan data pemenang ke Divisi Humas untuk bahan konten media.
